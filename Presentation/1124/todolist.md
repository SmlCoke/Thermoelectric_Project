# ✅ **任务 1：8 个通道的全天（12:10–17:08）原始电压时序图（最核心）**

**展示目的：**

* 展示采集系统稳定性
* 展示太阳辐射变化与时间关系
* 展示各波长通道的响应差异（短波/长波高低不同）

**图像形式：**

* 在同一张图里画 8 条曲线（透明度低一点）
* 时间轴：12:00–17:10
* 电压单位：mV

**你可以强调：**

* 数据连续，无明显断点
* 曲线随太阳角度下降而下降
* 各波段（热电芯片+滤光片）对不同波长响应不同

（这是一张“必备图”。）

# ✅ **任务 2：8 通道的平滑曲线（移动平均过滤：N=31 或 N=101）**

**展示目的：**

* 降噪
* 更容易看出趋势
* 便于观察波段响应差异

**方法：**
对每个通道做简单滑动平均：

```python
df['ch1_ma'] = df['ch1'].rolling(101).mean()
```

**展示重点：**

* 滤光片波段越靠近可见光/短波：峰值通常更明显
* 长波通道上升/下降更缓

# ✅ **任务 3：强度衰减率比较（下午时段 15:00–17:00）**

**可以计算：**

> 每个通道
> **衰减率 = (V_15:00 − V_17:00) / V_15:00**

画成柱状图。

**展示目的：**

* 反映不同波段对太阳高度角降低的敏感性
* 直观展示“短波衰减更快、长波衰减更慢”的趋势
  （如果滤光片顺序正确的话）

这是一张“物理意义很明确”的图，报告中非常好用。

# ✅ **任务 4：全天信号的归一化对比（0~1）**

对每个通道做：

```python
norm = (V - V.min()) / (V.max() - V.min())
```

然后画 8 条归一化曲线。

**展示目的：**

* 不关心总量，只关心响应形状
* 能看出不同波段峰值时间是否相同
* 若波段相近的通道曲线更接近，这是**滤光片组装正确**的证据

**非常适合做“系统验证”内容。**

# ✅ **任务 5：电压随太阳高度角（或时间）的简单拟合（指数/多项式）**

即使没有真实太阳高度角，也可以用时间代替。

例如：
对每个通道做拟合：

```
V(t) ~ a * exp(b*t)
或者  
V(t) ~ at^2 + bt + c
```

展示：

* 拟合曲线与实际数据吻合程度
* 各通道参数差异（可解释为对太阳辐射的敏感度不同）

**这是“数学建模味道”比较强的展示，适合周一汇报。**

# ✅ **任务 6：通道之间的相关性矩阵（Correlation Matrix）**

计算 8 个通道之间的皮尔逊相关系数，并做热力图。

**展示目的：**

* 同一段波长范围应该高度相关
* 极端不同波段的通道相关性应更低
* 可以用来判断通道工作是否正常、滤光片是否装反

**这是一个很漂亮、很“数据分析范”的图。**
