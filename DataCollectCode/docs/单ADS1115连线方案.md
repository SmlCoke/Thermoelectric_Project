# 单ADS1115连线方案

本方案使用1个ADS1115采集2组差分电压信号，连接2个TEC1-04906热电芯片。

## 硬件清单

- Raspberry Pi 5 × 1
- ADS1115 ADC模块 × 1
- TEC1-04906热电芯片 × 2
- 杜邦线（公对母、母对母）若干

## 一、ADS1115到Raspberry Pi 5的连线

ADS1115通过I2C总线与Raspberry Pi 5通信，需要连接4根线：

| ADS1115引脚 | Raspberry Pi 5引脚 | 说明 |
|------------|------------------|------|
| VDD | Pin 1 (3.3V) | 电源正极，使用3.3V供电 |
| GND | Pin 6 (GND) | 电源地线 |
| SDA | Pin 3 (GPIO 2 / SDA) | I2C数据线 |
| SCL | Pin 5 (GPIO 3 / SCL) | I2C时钟线 |
| ADDR | GND | I2C地址设置（接GND时地址为0x48） |

### 连线说明：
1. **VDD → Pin 1 (3.3V)**：为ADS1115提供3.3V电源
2. **GND → Pin 6 (GND)**：共地连接，确保信号参考一致
3. **SDA → Pin 3 (GPIO 2)**：I2C数据线，用于数据传输
4. **SCL → Pin 5 (GPIO 3)**：I2C时钟线，用于同步通信
5. **ADDR → GND**：设置I2C地址为0x48（默认地址）

## 二、TEC1-04906热电芯片到ADS1115的连线

使用差分测量模式，可以有效抑制共模噪声，提高测量精度。

### 第一个TEC芯片（通道0-1）：

| TEC1-04906引脚 | ADS1115引脚 | 说明 |
|---------------|------------|------|
| P脚（正极） | A0 | 差分通道正端 |
| N脚（负极） | A1 | 差分通道负端 |

### 第二个TEC芯片（通道2-3）：

| TEC1-04906引脚 | ADS1115引脚 | 说明 |
|---------------|------------|------|
| P脚（正极） | A2 | 差分通道正端 |
| N脚（负极） | A3 | 差分通道负端 |

### 连线说明：
1. 每个TEC芯片的P脚（正极）连接到ADS1115的一个通道
2. 对应的N脚（负极）连接到相邻的下一个通道
3. ADS1115会测量A0-A1之间的电压差（第一个芯片）
4. ADS1115会测量A2-A3之间的电压差（第二个芯片）

## 三、完整连线图

```
TEC芯片1                    ADS1115                 Raspberry Pi 5
┌─────────┐                ┌─────────┐              ┌──────────────┐
│   P脚   │───────────────→│   A0    │              │              │
│         │                │         │              │ Pin 1 (3.3V) │←───VDD
│   N脚   │───────────────→│   A1    │              │              │
└─────────┘                │         │              │ Pin 3 (SDA)  │←───SDA
                           │         │              │              │
TEC芯片2                    │         │              │ Pin 5 (SCL)  │←───SCL
┌─────────┐                │         │              │              │
│   P脚   │───────────────→│   A2    │              │ Pin 6 (GND)  │←───GND
│         │                │         │              │              │
│   N脚   │───────────────→│   A3    │              └──────────────┘

                           └─────────┘
```

## 四、注意事项

1. **电源电压**：Raspberry Pi 5提供3.3V和5V两种电压，ADS1115使用3.3V更安全稳定
2. **I2C地址**：ADDR引脚接GND时，ADS1115的I2C地址为0x48
3. **差分测量**：使用差分模式可以有效抑制共模噪声，提高微弱信号的测量精度
4. **增益设置**：在软件中需要设置合适的增益，推荐使用GAIN=16（±0.256V量程），适合TEC芯片的mV级信号
5. **接线顺序**：建议先断电连接所有线路，确认无误后再上电
6. **杜邦线质量**：使用质量好的杜邦线，避免接触不良导致数据异常

## 五、测试验证

连线完成后，可以使用以下命令测试I2C设备是否正常识别：

```bash
sudo i2cdetect -y 1
```

应该能看到地址0x48处有设备响应。
