## readme
本文件夹中的降采样数据集对应的降采样因子都是 6。
对应着我们报告和最终汇报（以及十五周汇报）中使用的降采样配置 sub6。

### 降采样示例
比如我们要对`[0,1,2,3,...99]` 执行$r=5$的降采样，就是将其分解为：
- 子序列1: `[0,5,10,15,...95]`
- 子序列2: `[1,6,11,16,...96]`
- 子序列3: `[2,7,12,17,...97]`
- 子序列4: `[3,8,13,18,...98]`
- 子序列5: `[4,9,14,19,...99]`

### 文件说明
- sub6： 降采样因子为 6 的降采样数据集文件夹，包含 6 个子序列的数据集，分别对应 6 种不同的起始点偏移量（off0~off5）。
- Yellow：黄光通道降采样后的图像，6张图片(off0~off5.svg)的起始数据点分别为第1个、第2个、第3个、第4个、第5个、第6个。
- Violet：紫光通道降采样后的图像，6张图片(off0~off5.svg)的起始数据点分别为第1个、第2个、第3个、第4个、第5个、第6个。
