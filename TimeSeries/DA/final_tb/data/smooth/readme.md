## readme
滑动平均算法主要由一个参数控制降噪力度：
1. smmoth_window: 异常值处理窗口长度，表示在检测异常值时，考虑前后多少个数据点进行均值计算。

在本文件夹中，针对数据集`data1220_1`算法降噪参数：，我们选取了三种不同的滑动窗口长度，分别为：
- `L=3`: 窗口长度为3的滑动平均降噪
- `L=5`: 窗口长度为5的滑动平均降噪
- `L=7`: 窗口长度为7的滑动平均降噪

分别得到以下降噪后的数据集：
- `data1220_1_L3.csv`: 使用窗口长度$L=3$进行降噪后的数据集
- `data1220_1_L5.csv`: 使用窗口长度$L=5$进行降噪后的数据集
- `data1220_1_L7.csv`: 使用窗口长度$L=7$进行降噪后的数据集
- `data1220_1.csv`: 原始数据集

三种数据集与原始数据集的对比图见：`Violet_smooth.svg`, `Yellow_smooth.svg`
其中，$L=5$的滑动平均方法，就是我们在报告和最终汇报（以及十五周汇报）中降噪配置 d1 的第二个组成部分。
